CRUD API

CRUD כתובת פרמטרים Body מה עושה איזה סוג שגיאות יכולות לחזור
GET /Api/user - - מחזירה את כל המשתמשים
GET /Api/user/:id id - מחזירה את המשתמש לפי id
POST /Api/login/user - Username, Password מחזירה את המשתמש לפי Username ו-Password
POST /Api/user - Role, Phone, Password, Email, Username הוספת משתמש
PUT /Api/user - Phone, Password, Email, Username עדכון פרטי משתמש
PUT /Api/updatePassword/user - Password עדכון סיסמא
GET /Api/product - - מחזירה את כל המוצרים
GET /Api/product?Category=:category Category - מחזירה את המוצר לפי Category
POST /Api/product - Phone, Address, Categories, State, Name הוספת מוצר
DELETE /Api/product/:id id - מחיקת מוצר לפי id
PUT /Api/product - Name או State או Categories עדכון פרטי מוצר
GET /Api/purchase - - מחזירה את כל ההזמנות
GET /Api/purchase/:id id - מחזירה את ההזמנה לפי id
GET /Api/byUserId/purchase/:userid Userid - מחזירה את ההזמנה לפי Userid
POST /Api/purchase - Product, userId הוספת הזמנה
